# Laravel-Ders-Yoklama-Sistemi
Bu proje, okul ortamlarında ders yoklamalarının dijital ortamda kolay ve hızlı bir şekilde alınmasını sağlamak amacıyla geliştirilmiş bir web tabanlı sistemdir.  Sistem, öğretmen ve öğrenci rollerine göre yetkilendirilmiş olup, kullanıcıların sisteme giriş yaparak kendi işlemlerini gerçekleştirmesine olanak tanır.
